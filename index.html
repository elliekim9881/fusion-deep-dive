✅ 좋은 프롬프트
당신은 10년 경력의 비서입니다. 아래 회의록을 임원 보고용으로 정리해 주세요. - 형식: 결정사항 3개, 액션아이템 5개 (담당자/기한) - 톤: 간결한 존댓말 - 길이: 한 페이지 이내 [회의록 본문 붙여넣기]
💡 프롬프트는 검색어가 아니라 "업무 지시서". AI를 신입사원이라 생각하고 일을 시키듯 쓰면 됩니다.

🧩

⚙️
System Prompt
"사용자 지정 지침"에 항상 적용될 기본 규칙을 저장. 한 번 만들면 평생 사용.

하네스 엔지니어링

⏱ 약 40분 · AI 에이전트 팀 설계
챗봇 vs 에이전트
챗봇 — 질문·답변 기계
말로 답 ("이거 뭐야?" → 알려줌)
한 번 요청, 한 번 응답
도구 없음, 메모리 없음
vs
에이전트 — 심부름꾼
행동으로 답 ("이거 처리해줘" → 해옴)
목표(Goal) · 도구(Tools)
메모리(Memory) · 행동(Action)
💡 "AI에게 묻는다"에서 "AI에게 시킨다"로 사고를 바꾸는 것이 에이전트 시대의 출발선
핵심 오케스트레이션 패턴 3가지
🔗
파이프라인
A → B → C
릴레이 경주. 순서가 중요한 작업.
예: 조사→초안→교정
🌐
팬아웃·팬인
1 → N → 1
분업 후 취합. 여러 주제를 병렬로 빠르게 처리
✅
생성·검증
생성 → 검토 ⟲
초안 후 품질 검수. 보고서·코드 등 품질 중요 작업

방법론 전문가 (5~6장)

↓
